***프로젝트 만들기 순서***

```swift
1. 프로젝트를 생성할 디렉토리에서 
$ cordova create hello com.example.hello "HelloWorld"
//hello : 폴더명
//com.example.hello : 번들식별자
//"HelloWorld": 파일명
$ cd hello
$ cordova platform add ios

2. 해당 디렉토리 내 프로젝트 실행
platform - ios - .xcodeproj 
```

***코드로바?***
